import機制的說明
--------------------------------------------------------------

這個機制提供了一種管理配置文件新方式, 讓您可以省去每次更新或升級服務端後,
都必須重新配置服務端conf文件的麻煩. 

我要怎樣才能使用此機制?
--------------------------------------------------------------

將你想要修改的設置項, 放到對應的import文件中即可.
舉個例子, 如果你想修改 /battle/exp.conf 中的一個設置項的值:

	// 基礎經驗值倍率 (百分比選項)
	base_exp_rate: 700

只需要將這個選項, 完整的複製到 /import/battle_conf.txt 中,
這樣就可以避免未來更新服務端帶來的配置問題.

很靈活, 不是麼?
